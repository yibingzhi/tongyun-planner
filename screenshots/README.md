# 截图目录

在此目录中添加应用截图，以便在 README 中展示。

## 建议的截图

1. **主界面** - 四象限矩阵视图
2. **列表视图** - 任务列表展示
3. **日历视图** - 日历界面
4. **设置页面** - 主题和配置选项
5. **桌面小组件** - 小组件窗口
6. **番茄钟** - 计时器界面

## 命名规范

建议使用以下命名格式：
- `matrix-view.png` - 矩阵视图
- `list-view.png` - 列表视图
- `calendar-view.png` - 日历视图
- `settings.png` - 设置页面
- `widget.png` - 桌面小组件
- `pomodoro.png` - 番茄钟

## 在 README 中使用

添加截图后，可以在 README.md 中使用以下语法：

```markdown
![截图描述](screenshots/screenshot-name.png)
```

例如：
```markdown
![四象限矩阵视图](screenshots/matrix-view.png)
```
