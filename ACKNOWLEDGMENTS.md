## 如何被添加到致谢列表

如果您为 QiYun List 项目做出了贡献，请通过以下方式联系我们：
1. 提交 Pull Request 更新本文件
2. 在 Issue 中 @mention 项目维护者
3. 通过邮件联系我们

### 贡献类型
- 代码贡献
- 文档贡献
- 翻译贡献
- 设计贡献
- 测试贡献
- 反馈贡献
- 社区贡献

### 贡献标准
- 代码贡献：提交过被合并的 Pull Request
- 文档贡献：改进过项目文档
- 翻译贡献：提供过翻译内容
- 设计贡献：提供过设计资源
- 测试贡献：报告过被确认的 Bug
- 反馈贡献：提供过有价值的反馈
- 社区贡献：帮助过其他用户
